Algoritmos

Para el analisis de algoritmo se buscan 3 cosas importantes que sea preciso, determinista y finito. Esto es porque debe averiguarse si los algoritmos son o no eficaces para su funcion, esto dependera por las entradas y las salidas del mismo.

Este ambito se calcula mediante medidas asintomaticas que son para ver que tan rápido crece el tiempo de ejecucion esto quiere decir si tardara mucho o poco tiempo el algoritmo, esto se basa en 3 notaciones. Para la elección de algoritmos de debe considerar los siguientes factores:

Complejidad temporal: Tiempo que tarda el algoritmo en ejecutarse

Complejidad espacial: Cantidad de memoria que se usa para completar

Complejidad del mejor caso: Cantidad minima de recursos usados para completarse

Complejidad del peor caso: La maxima cantidad de recursos que se usaran

Complejodad promedio La cantidad promedia de recursos que se usaran

Medicion del algoritmo

Estos se pueden analizar y probar de 2 formas:

Analisis empírico: Se prueba a solo ejecución de la aplicación

Analisis analítico: Se analisa en base al analisis del algoritmo
